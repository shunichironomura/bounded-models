# Installation

## Requirements

- Python 3.13 or later

## Using pip

```bash
pip install bounded-models
```

## Using uv (recommended)

If you're using [uv](https://docs.astral.sh/uv/), you can add bounded-models to your project:

```bash
uv add bounded-models
```

## Verify Installation

Check that bounded-models is installed correctly:

```python
import bounded_models
print(bounded_models.__version__)
```

## Next Steps

Now that you have bounded-models installed, proceed to the [User Guide](guide/index.md) to learn how to use it.
